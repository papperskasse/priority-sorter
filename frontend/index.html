ü§ñ AI-Powered Data Migration

If you have your to-do list in another format (Word, PDF, Note app), you can use AI to quickly migrate it here:

  1. Create a dummy task in this app and Export JSON.
  2. Upload that JSON file as a template to an AI (like ChatGPT or Gemini).
  3. Paste your old to-do data into the AI chat.
  4. Ask the AI to: "Convert my to-do list into a JSON file following the structure of the provided template."
  5. Download the result and Import it here!

üîÑ Using Multiple Devices

  1. Export JSON from Device 1
  2. Transfer file (email, USB, cloud drive, etc.)
  3. Import JSON on Device 2
  4. Your tasks are now on both devices!
